# SimpleDeploy Recipes

Community-contributed deploy recipes for [SimpleDeploy](https://github.com/vazra/simpledeploy).

Each recipe is a paste-ready Docker Compose stack with `simpledeploy.*` labels for endpoints, healthchecks, and resource limits.

## Browse recipes

- In SimpleDeploy: deploy wizard then "Browse community recipes".
- Online catalog: https://vazra.github.io/simpledeploy-recipes/
